## 📌 Overview

Cairo Duplicate Finder helps you find and remove duplicate files on your
computer easily, freeing up storage space and keeping your files organized.
The program uses smart scanning to analyze file content and automatically
detect duplicates — whether documents, images, or multimedia files —
while maintaining system performance.

---

## ✨ Features

- ✅ Professional, intuitive, and user-friendly interface
- ✅ Quick duplicate scanning using name and size comparison
- ✅ Deep content scan using MD5 hashing for 100% accuracy
- ✅ Detailed statistics on duplicates and wasted space
- ✅ Clear distinction between original and duplicate files
- ✅ Safe deletion with multiple confirmations to prevent accidental removal
- ✅ Batch processing — manage multiple files at once
- ✅ Optional subdirectory scanning
- ✅ Ability to stop scanning anytime during process
- ✅ Comprehensive keyboard shortcuts (F5, Esc, Del)
- ✅ 100% Free — no fees, subscriptions, or limitations

## 🚀 How to Use

1. **Download and Install** the software on your device
2. **Open the software** and click **Browse** to select a folder
3. **Choose Scan Type** — Quick Scan or Deep Scan
4. **Set Scan Options** — include subdirectories or exclude file types
5. **Click Start Scan** or press **F5** to begin scanning
6. **Monitor Progress** — stop anytime using the Stop button or **Esc**
7. **Review Results** — view duplicates with size, location, and details
8. **Select Files** for deletion while preserving important originals
9. **Confirm Deletion** through multiple safety confirmation windows
10. **Verify Results** — check freed storage space after cleanup
